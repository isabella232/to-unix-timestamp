# to-unix-timestamp

  Convert a date to a Unix timestamp.

## Installation

    $ component install segmentio/to-unix-timestamp

## Example

```js
var unix = require('to-unix-timestamp');

unix(new Date());
// 1380249249
```

## API

### toUnixTimestamp(date)
  
  Returns the Unix timestamp of a `date`.
